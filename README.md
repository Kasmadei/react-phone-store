# Simple React-app 
Store shop with PayPal Payment
Demo: https://kasmadei-phone-shop.netlify.com

## Installation

Use the package manager NPM to install app.

```bash
git clone https://github.com/Kasmadei/react-phone-store.git
npm install
npm start
```
